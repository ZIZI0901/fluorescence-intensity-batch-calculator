# 荧光图像批量光强值计算

在物理、生物等领域的实验研究中，常需通过荧光图像的灰度值来量化光强信息，以进行后续的数据分析。针对现有通用软件在处理大批量图像时操作繁琐、效率较低的问题，开发此工具以实现批量图像光强值的快速计算与结果导出。用户仅需指定图像文件夹路径并设置相应参数，即可自动完成所有图像的光强分析，并生成格式规范的Excel表格，便于后续统计与可视化处理。

---

## 使用步骤

1. **准备图像数据**：将所有待分析的荧光图像（支持JPG格式）置于同一文件夹下。

2. **选择图像目录**：启动程序后，通过界面选择或输入上述图像所在的文件夹路径。

3. **设置分析阈值**：输入灰度阈值。程序将仅对图像中灰度值**大于**此阈值的像素点进行计算，以排除背景噪声或非特异性信号的干扰。

4. **执行批量处理**：点击“处理图像”按钮，程序将自动遍历文件夹内所有图像，进行光强值计算，并在完成后将结果保存为Excel文件。
   
   

---

## 输出说明

生成的Excel文件包含以下字段，便于用户进行定量比较与分析：

* **图像名称**：显示对应的图像文件名（不包含文件扩展名），便于数据溯源。

* **最大值**：单张图像中所有像素点的最大灰度值，反映图像中最强的光信号峰值。

* **总光强**：图像中所有**高于设定阈值**的像素点的灰度值总和，用于表征目标区域的整体发光强度。

* **平均光强**：总光强除以图像的总像素点数（即图像分辨率，如1920×1080对应的像素总数）。该值提供了一个归一化的光强水平，可用于比较不同尺寸或分辨率图像的相对亮度。
